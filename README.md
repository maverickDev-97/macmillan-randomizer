# App Summary

Improve your English vocabulary by using this app. Get a random unknown word from 2,500+ words, navigate to the Cambridge dictionary to learn the definition of this word and usage examples. You can safely close the app if you've saved the words and come back to learn them later.

[Live Demo](https://macmillan-randomizer.onrender.com/)

## Technologies

This project was bootstrapped with Create React App and then migrated to the Vite build tool

- Typescript
- SCSS
